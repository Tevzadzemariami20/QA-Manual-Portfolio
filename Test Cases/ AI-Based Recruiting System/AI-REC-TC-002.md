# AI-REC-TC-002 - Verify Terms of Service on Login Modal
#### ID-Login-003
#### Priority: High
#### Test Type: Functional
#### Feature: Login Page
#### Test Status: Passed


## Description
- Verify that the Terms of Service link is visible on the Login modal, opens correctly and its content is accessible and readable.

## Pre-Conditions
- User has internet access
- Website is available

## Test Steps
| Step No. | Action	| Input Data	| Expected Result |
| -------- | -----	| -----------	| --------------- |
| 1 | Navigate to the website	| https://getbot.ai/	| Website loads successfully |
| 2 | Click on the Login button	| N/A	| Login modal opens |
| 3 | Verify the Terms of Service link is visible below the email input field	| N/A	| Terms of Service link is visible |
| 4 | Click on the Terms of Service link	| N/A	| Terms of Service page opens |
| 5 | Verify the Terms of Service content	| N/A	| Content is accessible, readable, and fully displayed |

## Post-Conditions
- User remains on the Terms of Service page or can return to the Login modal
