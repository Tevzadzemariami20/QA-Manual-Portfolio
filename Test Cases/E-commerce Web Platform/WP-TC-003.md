# WP-TC-003 – Change Language to English

#### Priority: Medium  
#### Type: Functional  
#### Feature: Language / Localization  
#### Status: Fail ❌  

## Description
- Verify that the user can change the website language to English using the language switch button.

## Pre-conditions
- User has access to the website
- Registration is not required

| Step No. | Action | Expected Result |
| -------- | ------ | --------------- |
| 1 | Open Website  | Website opens successfully and user is on the home page |
| 2 | Click the **English** button in the upper right corner | Website language changes to English |

## Actual Result
- The language does not change after clicking the English button.

## Post-conditions
- User remains on the same page
- Website language stays unchanged
