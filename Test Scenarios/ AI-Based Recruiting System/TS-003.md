### Goal:
Verify that the user can open a chat from a chat group and view chat content.

### Preconditions:
- User is registered
- User is logged in
- User has access to chat groups

### Flow:

- User opens the application
- User navigates to the chat dashboard
- User expands a chat group
- User selects a chat from the group
- User verifies that the selected chat opens successfully
- User verifies that chat messages are visible
