## TS-004 â€“ Submit Prompt Using Send Button

### Goal:
Verify that the user can submit a prompt using the Send button and see the message appear in the chat.

### Preconditions:
- User is registered
- User is logged in
- User has access to the chat dashboard

### Flow:
- User opens the website
- User logs into the application
- User navigates to the chat dashboard
- User enters a prompt into the chat input field
- User clicks the Send button
- System submits the prompt
- User verifies that the message appears in the chat without errors
