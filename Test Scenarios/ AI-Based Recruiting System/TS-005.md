## TS-005 â€“ Prevent Sending Empty Prompt

### Goal:
Verify that the system prevents users from sending an empty prompt using either the Send button or the Enter key.

### Preconditions:
- User is registered
- User is logged in
- User has access to the chat dashboard

### Flow:
- User opens the website
- User logs into the application
- User navigates to the chat dashboard
- User leaves the chat input field empty
- User clicks the Send button
- User presses the Enter key
- System blocks submission of the empty prompt
- User verifies that no message appears in the chat
