## TS-001 â€“ Add Product to Wishlist

### Goal:
Verify that a logged-in user can add a product to the wishlist and view it in the wishlist page.

### Preconditions:
- User is registered
- User is logged in
- Website is accessible

### Flow:
- User opens the website
- User navigates to a product category
- User selects a product
- User clicks the wishlist (heart) icon
- System adds the product to the wishlist
- User opens the wishlist page
- User verifies the selected product is displayed
