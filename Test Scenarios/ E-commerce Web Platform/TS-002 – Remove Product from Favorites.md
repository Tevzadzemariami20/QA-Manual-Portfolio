### Goal:
Verify that a user can remove a product from the Favorites list.

### Preconditions:
- User is registered
- User is logged in
- At least one product exists in Favorites

### Flow:
- User opens the Favorites page
- User identifies a product in the list
- User clicks the remove / unfavorite icon
- System removes the product from Favorites
- User verifies the Favorites list updates correctly
